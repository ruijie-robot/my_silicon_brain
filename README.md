# 项目要求

# 知识库
把documents文件夹下面的文件，只要有更新的，就实时的加入到milvus的vectorStore里面，供之后的RAG调用

# MCP tools
建立一个可以网络搜索的MCP Tools，以供LLM调用

# LLM大模型
下载一个开源的（要么来自Qwen，要么来自DeepSeek），适合4090性能的LLM大模型（并且可以接入MCP tools），在本地调用，用于回答金融类的问题，比如说，

- example 1: 这会儿市场上某个板块起来了，我去问，它能把这个版块里我相对比较熟悉的票给出来。

- example 2: 比如写完了复盘和计划，扔给它检查一下，今天还跟人聊了哪些机会没复盘到，或者计划里面哪些操作是违背了我的交易原则之类的

# 深度思考
使用LangChain进行深度思考的编排，并且对memory进行管理
